Where do I start from?

The first thing you need to do is add some items you're using often. You can do that through the {{lang.addNewItem}} screen.
These items will be presented in the initial screen and they will be easily accessible. For that reason, only frequently used items should be added so that the initial screen is not cluttered.

Adding new items

When adding a new item you can specify the following details:
Name
This is a required field. It must describe the item, ideally not with a broad term. For example, if you buy a lot of Coca-Cola then the name of the item should be 'Coca-Cola' and not 'Soft Drink'.
Description
An optional description for the item.
Price
An optional price for the item.
Some items have fixed price, for example your favorite tobacco. In these cases it's good to specify the price so that you don't have to enter it every time you click on the item.
If there is no standard price for the item, then this field should be left blank. This way, when you click on the item, ExpenseMe will ask you for the price you paid that particular time.
Category
Optional field which allows some grouping of items. This determine the icon that will be used for the item's button and also for later processing of the expenses, for example if you want to see how much you are spending on drinks in general and not only on Coca-Cola.
If no category is specified, then the first letter of the item's name will be used for the item's button.

Can I control the color of the item's button?

No, ExpenseMe does not allow you to specify a particular color for an item. The color will be randomly selected from a pool of 5 colors upon creation of the item.
However, an item will always have the same color. The only way to get a different color for an item is by deleting the item and adding it again.

How can I change the details of an item?

Items can be edited through the {{lang.editItems}} screen. You can change all the details of an item except from his name.
Changes to the details of an item will not affect previous uses (expenses) of this particular item. They will only affect future ones.
